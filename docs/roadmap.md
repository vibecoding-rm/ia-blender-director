# Roadmap

## Semana 1: Base Ejecutable

- Repo Git local.
- CLI.
- Esquema `ShotSpec`.
- Script Blender simple.
- Tests de validacion.

## Semana 2: Escenas y Camaras

- Mas primitivas.
- Movimientos de camara parametrizados.
- Luces predefinidas.
- Render de previews.

## Semana 3: Generador de Specs

- Prompt a JSON mediante reglas locales.
- Plantillas de escenas.
- Validacion estricta antes de ejecutar Blender.

## Semana 4: Personajes

- Carpeta de personajes versionados.
- Integracion con rigs fijos.
- Importacion de animaciones FBX/BVH.

## Semana 5: Render Passes

- Beauty.
- Depth.
- Mask.
- Normal.
- OpenPose o pose proxy si aplica.

## Semana 6: ComfyUI

- Primer workflow image-to-video o stylization.
- Script para llamar API local de ComfyUI.
- Versionado de workflows.

## Semana 7: Vision Critic

- Analisis de preview.
- Reglas de correccion simples: camara lejos, sujeto oscuro, encuadre debil.

## Semana 8: Interfaz Web

- UI minima para crear plano, renderizar preview y aprobar render final.
